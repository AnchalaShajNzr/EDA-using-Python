# EDA-using-Python
This project focuses on performing Exploratory Data Analysis (EDA) on customer data using Python.
EDA helps uncover patterns, insights, and relationships within the data, providing a foundation for decision-making and predictive modeling. Key features of this repository include:

Data Overview: Summarizing key statistics and understanding data structure.
Visualization: Creating insightful plots using Matplotlib and Seaborn for trends, distributions, and correlations.
Customer Segmentation: Identifying patterns and clusters using demographic and behavioral data.
Handling Missing and Outlier Data: Strategies for managing anomalies and incomplete records.
Feature Analysis: Evaluating numerical and categorical variables to determine their influence on business metrics.
